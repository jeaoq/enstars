# Oi~ssu!

Oissu (stylized as Oi\~ssu♪) is a script for formatting dialogue-based stories on Tumblr, mainly focused on delivering a better experience for game translations.

:::info
To get started using Oissu, click on **Installation** on the end of the page! Use the buttons to navigate the guide, or use the sidebar to skip to any part.
:::
